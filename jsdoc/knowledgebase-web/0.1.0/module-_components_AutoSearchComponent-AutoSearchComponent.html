Class: AutoSearchComponent

/components/AutoSearchComponent~AutoSearchComponent()

Autocomplete search component for querying ease of use. Text input component that makes small (default limit 30) calls to the api when the user types. Includes a debounce time to prevent large volumes of api calls. Results are displayed in a menu anchored below the input text field, inheriting its width and with a max-height proportional to the specified limit.
